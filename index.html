🔥 Connect Firebase
1
Create a Firebase project
Go to console.firebase.google.com → Add project → name it anything (e.g. my-journal) → skip Analytics → Create.
2
Add a Web App
In your project → click the </> (Web) icon → register app → copy the firebaseConfig object that appears.
3
Enable Firestore
In Firebase console → Firestore Database → Create database → choose Start in test mode → pick any region → Done.
4
Paste your config into this file
Open this HTML file in a text editor. Find the section near the top that says PASTE YOUR FIREBASE CONFIG HERE and replace the placeholder values with your real config. Save and reopen the file.
5
Host it (free, forever accessible by link)
In Firebase console → Hosting → Get started. Install Firebase CLI: npm install -g firebase-tools, then firebase login, firebase init hosting, put your HTML in the public/ folder, run firebase deploy. You'll get a permanent *.web.app link.
